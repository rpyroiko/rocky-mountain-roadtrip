# Rocky Mountain Roadtrip

## Overview
A planning tool for road trips in the Rocky Mountains, including weather documentation.

## Features
- Trip itinerary planning
- Weather documentation
- Points of interest

## Tech Stack
- Python
- Pandas

## Usage
Run the main script to get started:
```bash
python trip_planner.py
```
